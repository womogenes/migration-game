# Migration game

Under development.

## Setup

1. `npm install`
2. `npm run dev`

## Project structure

Built with [SvelteKit](https://kit.svelte.dev/).

- `src/routes/+page.svelte` contains all content for now.
- `lib/store.js` contains logic for making Svelte stores that sync with LocalStorage.
- `lib/locations.json` contains data about different locations (ambient messages, currency, etc.)
